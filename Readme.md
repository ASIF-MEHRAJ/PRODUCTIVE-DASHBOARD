🐼 Productivity Dashboard
A single-page productivity web app that brings together multiple daily-use tools into one clean, unified dashboard — helping users manage tasks, plan their day, stay motivated, track focus time, and check the weather, all without leaving a single screen.

Built entirely with vanilla HTML, CSS, and JavaScript — no frameworks, no libraries — to demonstrate core web development fundamentals: DOM manipulation, event handling, browser storage, and API integration.

✨ Features
📝 Todo List — Add, complete, and delete tasks with persistent storage
📅 Daily Planner — Organize your day into hourly time blocks
🎯 Daily Goals — Set short-term goals and track completion progress
⏱️ Pomodoro Timer — Stay focused with a customizable work/break countdown
💬 Motivation Quotes — Fetch fresh, inspiring quotes live from an API
🌤️ Weather Widget — Real-time weather updates based on location
🕐 Live Date & Time — Always-visible, auto-updating clock and date
🌗 Light/Dark Theme — Toggle between themes, remembered across sessions
🎨 Dynamic Background — Interface adapts subtly based on time of day

🛠️ Built With
HTML5 — Semantic structure for the dashboard and all feature sections
CSS3 — Custom properties (CSS variables) for theming, responsive layout, animations
JavaScript (ES6) — DOM manipulation, event delegation, Fetch API, timers
Local Storage — Persists tasks, goals, planner entries, and theme preference across sessions
Public APIs — Live quote and weather data

🚀 How It Works
Dashboard loads with quick-glance widgets (time, weather) and feature cards
Click any card to open that feature in full view
Interact with the feature — add tasks, start a timer, mark goals complete
Data is automatically saved to Local Storage
Return to the dashboard anytime with a single click
